# Ana V style

## Core voice
- Polite, businesslike, direct
- Friendly but not chatty
- Usually warm-neutral rather than playful
- Uses short practical replies in ongoing logistics
- Uses longer promo-style intros for cold outreach

## Typical structure

### Cold outreach
Usually includes:
1. greeting + name
2. travel window / city availability
3. short value pitch
4. invitation to shoot together
5. sign-off

### Negotiation / logistics
Usually includes:
1. direct answer first
2. rate / date / condition
3. 1-3 practical follow-up questions
4. short sign-off

## Language behavior
- Tends to reply in the interlocutor's language
- Often mirrors German with German, English with English
- Can switch to simple international English for logistics

## Tone markers
- "It would be a great pleasure for me to work together"
- "Have a great day"
- "Best wishes, Ana!"
- "LG Ana" / "Best regards, Ana"
- "Hello, yes..." / "Hi..." / "Hallo..."

## Style characteristics
- Prefers clarity over charm
- Often answers with concrete booking details
- Comfortable listing rates and levels explicitly
- Does not over-explain emotions
- Uses short confirmation messages when conversation is already established

## Weaknesses in existing style
- Some repeated boilerplate in outreach
- Occasional unnatural phrasing / grammar
- Sometimes too abrupt in very short replies
- Sometimes includes too much pasted rate-card text when a shorter answer would do
